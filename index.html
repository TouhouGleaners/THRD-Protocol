    欢迎来到拾遗梦茶馆

    THRD 是什么

    拾遗梦茶馆（Teahouse of Recollected Dreams） 是一个专注于东方 Project 资源归档与保存的社团。我们做的事情很简单：备份那些可能会消失的视频、弹幕和记忆。

    茶馆的核心理念是"管理即代码"——我们用 GitHub 来管理组织，用 PR 来做决策，用文档来记录规则。这样做的好处是：所有决定都有迹可循，不会变成一言堂。

    从哪里开始

    作为茶馆的维护者，你需要了解以下内容。推荐按顺序阅读：

    • 改革白皮书（必读）——了解茶馆为什么要改革，这是理解后续所有文档的基础。
    • 角色定义（必读）——了解三类维护者的职责与边界。
    • 决策流程（必读）——了解茶馆如何通过 PR 做出决策。
    • 技术大纲（推荐）——了解各职能路径的技术要求。
    • THRD 精神（推荐）——了解茶馆的价值观与行事准则。

    如果你还没有完成入馆流程，请阅读入馆指南，它会带你完成第一个 PR。
