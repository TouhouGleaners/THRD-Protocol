# THRD 精神：维护者共识 (Common Consensus)

这份文档不代表任何个人的意志，而是尝试总结“拾遗梦茶馆”自成立以来，支撑我们共同走下去的那些朴素共识。  
我们将其记录下来，是为了让未来的协作有据可依。

---

## 1. 我们为什么在这里？
大家聚在一起，初心很简单：**不希望那些喜欢的东方 Project 资源（视频、弹幕、记忆）因为各种意外而彻底消失。**

- 我们是“拾荒者”：在数据的废墟里寻找有价值的碎片。
- 我们是“守望者”：通过备份和整理，延长这些记忆的寿命。

## 2. 我们的处事默契
这些是我们过去曾尝试探索的道路，现在我们尝试将其标准化：

- **非营利与自发**：我们基于热爱而协作，不涉及商业利益，也不希望被商业逻辑束缚。
- **尊重原作者**：补档的目的是保存，而不是盗用。在任何时候，我们都应尊重并标注原作者的劳动成果。
- **极简高效**：能用一页文档说清楚的事，不开会；能用一个脚本解决的事，不堆人力。
- **互相补位**：我们意识到“单点故障”是组织最大的风险。学习技术、完善文档，本质上是为了在同伴忙碌时，其他人能顶得上去。

## 3. 关于“维护者”的自律
既然我们选择了这个身份，就意味着我们达成了一组默契：

- **数据真实**：不对归档数据进行恶意篡改。
- **公开透明**：组织的规则（如这份 Protocol）对全员开放，每个人都有权通过 PR 提出修改建议。
- **低压协作**：茶馆是大家忙碌生活外的避风港，不强制贡献，不给同伴制造额外的心理压力。

---

## 4. 这取决于我们每个人

这份文档是动态的。如果随着时间推移，你觉得茶馆的精神发生了变化，或者有更好的想法，请随时发起 PR。**茶馆的未来不属于某一个人，而属于每一位留下提交记录（Commit）的维护者。**
