# 欢迎来到拾遗梦茶馆

## THRD 是什么

**拾遗梦茶馆（Teahouse of Recollected Dreams）** 是一个专注于东方 Project 资源归档与保存的社团。我们做的事情很简单：备份那些可能会消失的视频、弹幕和记忆。

茶馆的核心理念是**"管理即代码"**——我们用 GitHub 来管理组织，用 PR 来做决策，用文档来记录规则。这样做的好处是：所有决定都有迹可循，不会变成一言堂。

## 怎么加入

如果你已经决定加入茶馆，请阅读[入馆指南](./onboarding.md)，它会带你完成第一个 PR。

- **想了解茶馆有哪些角色？** → 阅读[角色定义](./maintainer-role.md)
- **想知道茶馆怎么做出决策？** → 阅读[决策流程](../action-guide/decision-making.md)
- **想学习技术技能？** → 阅读[技术大纲](../skills/tech-stack.md)
