# 欢迎来到拾遗梦茶馆

## THRD 是什么

**拾遗梦茶馆（Teahouse of Recollected Dreams）** 是一个专注于东方 Project 资源归档与保存的社团。我们做的事情很简单：备份那些可能会消失的视频、弹幕和记忆。

茶馆的核心理念是**"管理即代码"**——我们用 GitHub 来管理组织，用 PR 来做决策，用文档来记录规则。这样做的好处是：所有决定都有迹可循，不会变成一言堂。
